## How to Get Your Video

### The Simplest Way (5 minutes)

```bash
# Step 1: Install Remotion
npm install remotion@latest

# Step 2: Preview (optional but recommended)
npm run remotion:preview
# Check it looks good in browser, then close

# Step 3: Render
npm run remotion:render
# Wait 5-7 minutes...

# Step 4: Find your video
# output.mp4 in your project folder
```

That's it! Your video is ready.

### With Music (15 minutes)

Same as above, but:

1. Download free music from Pixabay or Incompetech (3 min)
2. Place in `remotion/assets/audio/background-music.mp3`
3. Uncomment Audio tag in `remotion/TeraDemo.tsx` (1 min)
4. Run render (5-7 min)

See **AUDIO_SETUP.md** for details.

### Different Quality/Size Options

```bash
# 4K (high quality, ~30 min rendering)
npx remotion render remotion/composition.ts TeraDemoVideo output-4k.mp4 \
  --width 3840 --height 2160

# Custom quality (0-100, default 80)
npx remotion render remotion/composition.ts TeraDemoVideo output.mp4 \
  --quality 95
```
