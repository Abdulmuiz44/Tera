================================================================================
                        RENDERING OPTIONS
================================================================================

BASIC (5-7 minutes):
  npm run remotion:render
  Result: output.mp4 (1920x1080, ~40MB)

4K (20-30 minutes):
  npx remotion render remotion/composition.ts TeraDemoVideo output-4k.mp4 \
    --width 3840 --height 2160
  Result: output-4k.mp4 (~200MB)

CUSTOM QUALITY (adjust rendering time):
  npx remotion render remotion/composition.ts TeraDemoVideo output.mp4 \
    --quality 95
  
  Quality: 0-100 (higher = better but slower)
  Default: 80

ProRes (for editing):
  npx remotion render remotion/composition.ts TeraDemoVideo output.mov \
    --codec prores-422
  Result: output.mov (uncompressed, huge file size)

================================================================================
                        CUSTOMIZATION
================================================================================

CHANGE COLORS:
  Edit: remotion/utils/colors.ts
  Variables: teraNeon, teraBg, teraPrimary, teraPanel, etc.

ADJUST TIMING:
  Edit: remotion/TeraDemo.tsx
  Change SCENE*_DURATION values (in frames, 30fps = 1 second)

MODIFY TEXT:
  Edit: remotion/scenes/Scene*.tsx files
  Find <h1>, <p>, <span> and update content

ADD AUDIO:
  1. Download MP3 from Pixabay or Incompetech
  2. Place in: remotion/assets/audio/background-music.mp3
  3. Uncomment <Audio> tag in TeraDemo.tsx
  4. Render

ADD LOGO/IMAGES:
  1. Place files in remotion/assets/
  2. Import in scene files
  3. Use <img> or in JSX

================================================================================
                        TIPS & TRICKS
================================================================================

BEST PRACTICES:
  - Preview before rendering (npm run remotion:preview)
  - Render at 1080p first, then 4K if needed
  - Use high-quality background music (120-130 BPM recommended)
  - Test output on phone, tablet, desktop
  - Add subtitles using CapCut (free) or DaVinci Resolve (free)
  - Create 15-second clips for TikTok/Reels
  - Embed on landing page hero section
  - Track views and engagement on YouTube

WARNINGS:
  - Don't use copyrighted music without permission
  - Don't render 4K unless you need it (slow, large files)
  - Test audio works before final render
  - Close other apps while rendering (frees up CPU)
